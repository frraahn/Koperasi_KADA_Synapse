# Koperasi KADA Automated System

## Project Overview
The **Koperasi KADA Automated System** is a web-based platform developed to streamline and digitize the membership, loan management, and financial workflows of the Lembaga Kemajuan Pertanian Kemubu (KADA) Cooperative. 

Historically dependent on slow, paper-based operations, the cooperative faced challenges regarding processing delays, record inaccuracies, data security, and manual loan calculations. This system addresses those issues by introducing end-to-end digital automation, replacing physical paperwork with secure online forms, centralized database tracking, real-time email notifications, and automated calculation modules.

## Core Modules & Features
* **Membership Application:** Digital registration and data submission for prospective members.
* **Terminate Membership:** Streamlined cancellation request flows for existing members.
* **Review Module:** A dedicated interface for system Administrators to inspect submissions for data completion.
* **Loan Application:** Allows members to simulate repayment timelines via an integrated loan calculator and submit digital credit requests.
* **Approve Application:** Permissive decision-making tools enabling the board members (Ahli Lembaga Koperasi - ALK) to approve or reject pending files.
* **Statement Report Production:** Generates clear, auditable individual statements and overarching cooperative financial records.
* **Financial Tracking:** Monitors historical data regarding ongoing member savings, transactions, and outstanding repayments.

## Tech Stack
* **Frontend:** HTML, CSS, JavaScript, Bootstrap 
* **Backend:** PHP 
* **Database Management:** MySQL, phpMyAdmin (via XAMPP local environment)
* **Hosting:** Jimat Hosting
